Basic Installation

Steps to be followed for getting started with the template:

There are followings basics packages you should install before going further.

  1. Install NodeJs from NodeJs Official Page on your computer
  2. Open terminal or CMD and go the root directory of the template
  3. Run in terminal or CMD: npm install
  4. To Run project on Local environment run: npm run dev
  5. Then visit: http://localhost:3000
  6. To Deploy project on server
    1. Install PM 2 run : sudo npm install pm2@latest -g
    2. Copy project in The root directory of the server
    3. cd project_directory
    4. Run command npm install
    5. Run command npm run build
    6. Run command pm2 start npm -- start
    7. For the detailed Information please refer this blog Setup a Next.js project with PM2,
  7. Now that you have followed these steps your basic installation is complete and ready to flaunt your site to the world.

For Changing Fonts

You can add/change the site font, from all fonts used from Google Web Font Services, with the one that suits you the best. You can find the font link in top of the Style.css in all HTML file. See example below:


<!-- Main style.css -->
| - xamin-react
    | - static
        - style.css  // file content

        <!-- Google fonts -->
        @import url("https://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700,800|Raleway:300,400,500,600,700,800,900");

To include new font you can simply add another link like this: 


<!-- Main style.css -->
| - xamin-react
    | - static
        - style.css  // file content

        <!-- Raleway font -->
        @import "https://fonts.googleapis.com/css?family=Raleway:100,100i,300,300i,400,400i,500,500i,700,700i,900,900i";

        <!-- Open Sans font -->
        @import "https://fonts.googleapis.com/css?family=Open+Sans:400,400i,600,700,800";

In order to change the fonts, you will need to edit the above links with your custom font, You can easily use Google Web Font Services if you plan to use a Google Font or remove it completely. If you plan to use a self hosted font, here is an Example of using Self Hosted Fonts

For section padding

You can add this helper class to set section padding top 100px and padding bottom 100px.
Add iq-ptb-100 class in section tag. See example below: 

<section className="... iq-pt-100">

[YOUR CONTENT]
</section>

For Background color

You can use color in the background. simply add .iq-bg-hover (or any color you want) class where you want to use. See example below: 

<div className=".iq-bg-hover">
    [YOUR CONTENT]</div>

Note We include 4 background color helper class in our template .iq-bg-hover You can add unlimited background color class according to your needs
